Estructura del proyecto
El Drive del proyecto está organizado en las siguientes carpetas, con el objetivo de
mantener una separación clara entre datos, código, documentación y resultados:
archive
Contiene los archivos por tipos de fruta utilizados para Machine Learning y Deep
Learning.
csv
Incluye las tablas en formato .CSV para su visualización, clasificadas por categoría cómo
por tipo de fruta.
ideas
Carpeta destinada a planteamientos iniciales, pruebas y errores con Google Collab.
json
Almacena archivos originales del proyecto en formato JSON.
ML y DL
Contiene material relacionado con Machine Learning y Deep Learning, incluyendo
modelos, scripts y comparación.
Notebooks
Incluye los Notebooks con el EDA, pipeline y análisis individual por cada tipo de fruta.
PBI
Contiene archivos y recursos de Power BI utilizados para la visualización de datos y
creación de dashboards.
pdfs
Documentación en formato PDF: Informe ejecutivo y caso de uso: IA Generativa.
sql
Scripts SQL utilizados para la creación, consulta y gestión de bases de datos.
